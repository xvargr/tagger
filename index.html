DRAC TAGGER

Select one or multiple .txt file and press start, this program should be able to extract useful data such as tags, as long as they are present in the file and consistent with your normal formatting

If a field is missing, it can be flagged as such and added manually later. This data can then be used to search and filter results to discover your stories

This is something I put together over a weekend as a proof of concept, just meant to get the point across and not very pretty
